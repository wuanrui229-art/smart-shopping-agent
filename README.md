# Smart Shopping AI

Smart Shopping AI 是一个支持开放式对话的智能购物助手。用户只需用自然语言描述想买的商品、预算、用途和偏好，系统就会理解需求，给出三款候选商品，并解释推荐理由、优缺点和购买注意事项。

它不局限于页面上的六个示例品类，也可以处理空气炸锅、化妆品、电脑、家电、服饰等其他消费品类。

## 在线体验

[打开 Smart Shopping AI 在线 Demo](https://smart-shopping-agent-wuanrui.vercel.app)

无需注册，打开后直接在输入框中描述需求即可。例如：

- `我想买一台适合两个人、预算 1000 元以内、容易清洗的空气炸锅`
- `推荐一款适合通勤、预算 500 元以内的防水双肩包`
- `I need running shoes for daily training`

如果条件不足，助手会继续询问预算、用途或偏好；信息足够后，它会生成 Top 3 推荐和横向对比。页面上的六个快捷按钮只是演示入口，不是品类限制。

## 主要功能

- 中英文开放式多轮对话
- 不限预设品类的购物需求理解与 Top 3 推荐
- 综合预算、用途、品牌和个人偏好进行决策
- 展示推荐理由、优缺点、多维评分和商品对比
- 保存本地会话记录和购物偏好
- 对内置商品数据进行多维评分和可解释排序

## 系统组成

- **前端**：聊天界面、偏好设置、推荐结果与对比展示
- **后端**：FastAPI 接口、实时流式响应、会话与推荐流程管理
- **智能算法**：Kimi 大语言模型负责理解用户需求并生成推荐，评分算法结合价格、需求匹配度和用户评价，对候选商品进行可解释排序

## 本地运行

需要 Python 3.9 或更高版本。

macOS / Linux：

```bash
chmod +x start.sh
./start.sh
```

Windows：

```bat
start.bat
```

启动后访问 [http://127.0.0.1:8000](http://127.0.0.1:8000)。

> 推荐结果以 Amazon 商品搜索链接作为购买入口，但系统未接入 Amazon 实时商品 API。商品价格、库存和评价可能发生变化，购买前请以 Amazon 实际页面为准。
